## PINN

В данной ветке реализовано обучение PINN, используемой для аппроксимации матричной экспоненты.

После обучения веса PINN сохраняются и затем используются в основной модели BuNN для быстрого вычисления матрицы преобразования.

Также в ветке приведена оценка качества PINN. Для проверки сравниваются истинные траектории с предсказаниями сети. Полученные результаты показывают, что PINN практически полностью воспроизводит исходные траектории, что позволяет использовать ее в качестве эффективной замены прямого вычисления матричной экспоненты.
