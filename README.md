# Оптимизация работы графовых нейронных сетей на пучках при помощи методов низкоразмерной параметризации ортогональных матриц

## Цель работы

В настоящее время обучение графовых нейронных сетей на пучках требует значительных вычислительных затрат из-за большого количества параметров. Поэтому целью данной работы является оптимизация такого подхода с использованием методов низкоразмерной параметризации.


# Инструкция по запуску

## 1. Клонирование репозитория

```bash
git clone https://github.com/XenaBel2004/Sheaves_GCN.git
cd Sheaves_GCN
```

## 2. Создание виртуального окружения

```bash
python -m venv .venv
```

Активируйте окружение:

**Linux/macOS**

```bash
source .venv/bin/activate
```

**Windows**

```bash
.venv\Scripts\activate
```

## 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 4. Запуск проекта

Проект запускается **в два этапа**.

### Шаг 1. Построение эмбеддингов

Сначала необходимо сгенерировать эмбеддинги:

```bash
python main_embed.py
```

На этом этапе создаются эмбеддинги узлов, которые затем используются основной моделью.

---

### Шаг 2. Обучение модели

После генерации эмбеддингов запустите обучение модели:

```bash
python main.py
```

Во время выполнения будут выведены:

- точность на обучающей выборке (`accs_train`);
- точность на валидационной выборке (`accs_valid`);
- точность на тестовой выборке (`accs_test`);
- общее время обучения модели (`Training time`).
